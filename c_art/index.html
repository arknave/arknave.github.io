Advent of Code 2020 in C

Calling this “art” might be a stretch, but I am proud of these programs! Here are my solutions in C to the first fifteen days of Advent of Code, shaped roughly like the fifteen non-zero hexadecimal digits. You can see my Python solutions which I used to place on the leaderboard and more information about how to run these programs on the GitHub repository. Clicking on the number in the header of each day links to the original problem statement. I had a lot of fun competing and creating, thank you so much to the Advent of Code team for putting this together each year!
